## About ##
Extreme Learning Machines (ELMs) are randomized feed-forward networks invented by G.B. Huang [1, 2]. The input and hidden layer weights remain untrained, unlike output layer weights. Output layer weights are determined with either linear regression or Mooreâ€“Penrose generalized inverse. This Python project trains ELM with linear regression models.
